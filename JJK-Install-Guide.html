Everything you need to get into the server. Follow the steps in order — it takes about 10 minutes the first time.

Before you start

Get these three things ready first.

  • 01SKlauncher installedThe launcher you already use to play.
  • 02The modpack zipNamed JJK-Modpack-1.20.1.zip — the host sends this to you. Don't unzip it yet.
  • 03The server addressThe host will give you an address to connect to. Keep it handy for the last step.

Install it

Five steps. Do them in order.

1

Create a Forge 1.20.1 install

Open SKlauncher → Installations Manager (left menu) → New Installation. Set it up exactly like this:

Name
JJK 1.20.1
Version type
Forge
Minecraft
1.20.1
Forge loader
47.4.0

Under Game Directory, point it at a new empty folder — for example C:\JJK. This keeps the 40 mods separate from your normal Minecraft so nothing breaks. Then click Save.

2

Launch once to finish Forge

Back on the main screen, select your JJK 1.20.1 install and click Play. Let it reach the title screen — this downloads Minecraft 1.20.1 and installs Forge. Then close the game.

iThis first launch has no mods yet — that's expected. You're just letting Forge set itself up.
3

Copy everything in

Unzip JJK-Modpack-1.20.1.zip and copy all of its contents into your game directory from Step 1 (e.g. C:\JJK), overwriting if asked. You get four things:

mods/
the 41 mods
options.txt
pre-set keybinds (no conflicts)
resourcepacks/
custom JujutsuCraft textures
shaderpacks/
Complementary shaders (optional)
!Use all 41 mods — exactly. Don't add or remove any. Everyone on the server must run the identical set, or you'll get kicked with a mod-mismatch error.
4

Turn on the extras (in-game)

Textures: Options → Resource Packs → enable Customized JujutsuCraft (sharper JJK models & textures).

Shaders (optional, needs a decent GPU): Options → Video Settings → Shader Packs → enable Complementary Reimagined. Turn it off here if your FPS drops.

Keybinds are already set up for you — no conflicts. Tweak any of them in Options → Controls if you like.

5

Launch and connect

Start SKlauncher, pick the JJK 1.20.1 install, and hit Play. First load is slow (40 mods) — give it a minute. At the main menu go to Multiplayer → Direct Connection and enter the address the host gave you:

Server address
whacking-prevent.gl.joinmc.link

Paste that exactly (no port needed) and hit Join Server. You're in.

If something goes wrong

The usual snags and their fixes.

Mods“needs language provider” or “failed to load”

Your mods folder is missing a file. Delete everything in it and re-copy all 40 jars from the zip. All of them have to be there.

Kicked“mod mismatch” when joining

You have a different mod set than the server. Make sure it's the exact 40 from the zip — nothing added, nothing missing, same versions.

Java“requires Java 17” / won't launch

SKlauncher usually handles Java automatically. If not, install Temurin (Adoptium) Java 17 and pick it in the install's settings.

ConnectCan't reach the server

Use the exact address the host gave you (not localhost). Double-check the host's server is actually on. If it still fails, the host may need to restart their tunnel.

LagLow FPS or stutter

The pack already includes performance mods. In the install's settings, give it 4–6 GB RAM if you have it, and it'll smooth out after the first few minutes.
